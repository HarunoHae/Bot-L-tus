# Política de Privacidade – [Bot Lótus]

Data de Criação: [07/04/2026]
Última atualização: [07/04/2026]

## 1. Introdução
Esta Política de Privacidade descreve como o bot **[Bot Lótus]** ("Lótus") coleta, utiliza e armazena informações ao ser utilizado na plataforma Discord.

Ao utilizar o Bot, você concorda com os termos desta Política.

---

## 2. Informações coletadas

O Bot pode coletar e processar as seguintes informações:

- ID de usuário do Discord
- ID de servidor (guild)
- ID de canal
- Nome de usuário (quando necessário para comandos)
- Dados de configuração definidos por servidores ou usuários

O Bot **não coleta dados pessoais sensíveis**, como:
- Senhas
- E-mails privados (a menos que você forneça voluntariamente)
- Dados fora do Discord

---

## 3. Conteúdo de mensagens

O Bot pode acessar mensagens enviadas em canais onde está presente **apenas quando necessário para executar comandos**.

O Bot:
- ❌ Não armazena mensagens privadas permanentemente  
- ❌ Não lê mensagens sem motivo funcional  
- ✔ Pode processar mensagens temporariamente para executar comandos  

---

## 4. Uso das informações

As informações coletadas são utilizadas exclusivamente para:

- Executar comandos do Bot
- Armazenar configurações de servidores
- Melhorar funcionalidades
- Prevenir abuso e uso indevido

---

## 5. Compartilhamento de dados

O Bot **não vende, aluga ou compartilha dados com terceiros**, exceto:

- Quando exigido por lei
- Para funcionamento técnico (ex: hospedagem, banco de dados)

---

## 6. Armazenamento de dados

Os dados são armazenados apenas pelo tempo necessário para o funcionamento do Bot.

Medidas razoáveis são tomadas para proteger as informações armazenadas.
